Polyclaw provides a rich set of capabilities beyond basic chat.

## Core Features

- [Messaging & Bot](/features/messaging/) -- Multi-channel messaging through Bot Framework
- [Realtime Voice](/features/voice/) -- Phone calls via Azure Communication Services
- [Scheduler & Proactive](/features/scheduler/) -- Cron tasks and autonomous messaging
- [Media Handling](/features/media/) -- File, image, audio, and video processing
- [Sandbox Execution](/features/sandbox/) -- Isolated code execution in Azure Container Apps
- [Memory System](/features/memory/) -- Automatic conversation consolidation
- [Slash Commands](/features/commands/) -- Comprehensive command interface
